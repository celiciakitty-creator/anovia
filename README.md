# Anovia

AI-powered project management platform focused on productivity,
motivation and healthy work habits.

## Live Demo

https://anovia.vercel.app

## Features

- Dashboard
- Projects
- Tasks
- Calendar
- Wellness Hub
- Break Zone
- Growth Garden
- Kizuna AI
- Themes
- Comments

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Vercel

## Setup

npm install

npm run dev

## Current Limitations

- localStorage persistence
- no authentication
- single-user
